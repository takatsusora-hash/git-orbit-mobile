# GitHub Setup

## Proposed Repository

- Name: `git-orbit-mobile`
- Visibility: `private`

## Why This Name

- `git`: clearly tied to GitHub development data
- `orbit`: fits the 3D visualization concept
- `mobile`: keeps the smartphone target explicit

## Create The Repo Manually

Because this environment does not currently have `git` or `gh` available, the repository needs to be created once in the GitHub UI.

1. Open GitHub and create a new repository.
2. Set the repository name to `git-orbit-mobile`.
3. Set visibility to `Private`.
4. Do not add a README, `.gitignore`, or license there.
5. Create the repository empty.
